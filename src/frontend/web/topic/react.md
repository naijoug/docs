## concept

- `HMR`: `Hot Module Replacement` - “热模块替换”
> 它允许在开发过程中，不刷新整个页面的情况下，动态地更新 `React` 组件。当代码发生变化时，`HMR` 可以将新的模块代码推送到客户端，并且只替换应用中改变的部分，从而提高开发效率，让开发者能够更快速地看到代码修改后的效果。
