## 三核心

### Row - “行”
> X 轴 : 水平布局

### Column - “列”
> Y 轴 : 垂直布局

### Stack - “栈”
> Z 轴 : 层叠布局

> Positioned : 位置组件，用于控制子组件在栈中的位置

- top、bottom、left、right
- width、height

## 容器

### Container - “容器”
> 大融合容器(尺寸、形状、颜色、间距...)
> 尺寸规则 : 无子组件，填充父组件；有子组件，包裹子组件。

- width : 容器宽度
- height : 容器高度
- color : 容器背景色
- alignment : 子组件对齐方式
- margin : 外边距
- padding : 内边距
- constraints : 约束
- decoration : 装饰器
- foregroundDecoration : 前景装饰器
- clipBehavior : 裁剪
- transform : 形变
