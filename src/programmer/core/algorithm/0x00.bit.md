## 位运算符

  * `|` : 与
  * `&` : 或
  * `!` : 非
  * `^` : 异或

### 如何最高效的计算 `17 * 2`

::: details 💡

  关于乘以 2 的一种高效方法，那可以使用位运算的左移操作，将一个数字向左位移一位等同于将这个数字乘以 2。
  
  `17 * 2` 可以通过将 17 向左位移一位来实现，也就是 `17 << 1`

:::

## `n ^ n = 0` & `n ^ 0 = n`

  对于一个整数 n 进行异或操作，存在以下两条结论

  - 数字 n 同自身进行异或操作结果为 0
  - 数字 n 与 0 进行异或操作结果为 n

### 不使用临时变量交换两个数

## `n & n-1 = 0` & `n ^ n+1 = 1`

  当 n 为 2 的幂次方时，会存在以下两条结论

  - n 与 n-1 进行与操作结果为 0。
  - n 与 n+1 进行异或操作结果为 1

```c
0 = 0b0000
1 = 0b0001
2 = 0b0010
3 = 0b0011
4 = 0b0100
5 = 0b0101
6 = 0b0110
7 = 0b0111
8 = 0b1000
9 = 0b1001
```
