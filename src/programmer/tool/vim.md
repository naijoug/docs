## Commands

  ```shell
  # Liunx
  $ sudo apt-get install vim  # 安装Vim
  $ sudo vim /etc/vim/vimrc   # 开启高亮，去掉 synax on 前面 "

  # macOS
  $ cp /usr/share/vim/vimrc ~/.vimrc
  $ vi ~/.vimrc               # 加上 syntax on (开启高亮) 和 set nu! (显示行号)
  ```
