## 0344

// #region 0344

- [🟢 344 - 反转字符串](https://leetcode.cn/problems/reverse-string/)
  > 给定一个字符数组，返回反转后的字符数组。

::: details 💡

```java
class Solution {
    public void reverseString(char[] s) {
        int left = 0;
        int right = s.length - 1;
        while ( left < right ) {
            // 交换左右位置值
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            // 继续跳到下一个位置进行处理
            left += 1;
            right -= 1;
        }
    }
}
```
