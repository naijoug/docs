## 冒泡排序和插入排序的时间复杂度是多少？=> 为什么实际开发中会更多选择插入排序？


::: details 💡

  两者的时间复杂度相同 O(n²)
  
  插入排序是稳定的排序，对于相同的元素排序前后的相对位置不会发生变化。

:::
