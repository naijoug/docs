| `boolean`     | 布尔 | 
| `number`      | 数字 | 
| `string`      | 字符串 |
| `array`       | 数组 |
| `enum`        | 枚举 |
| `tulpe`       | 元组 |
| `object`      | 对象类型 |
| `any`         | 任意类型 |
| `void`        | 空类型 |
| `null`        | 空 |

| `unknown`     | 未知 |
| `never`       | 永不存在 |
