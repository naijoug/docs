## 数据类型一览

| 类型 | 说明 | 定义
| -- | -- | --
| `boolean`     | 布尔 | 
| `number`      | 数字 | 
| `string`      | 字符串 |
| `array`       | 数组 |
| `any`         | 任意类型 |
| `enum`        | 枚举 |
| `tuple`       | 元组 |
| `object`      | 对象类型 |
| `void`        | 空类型 |
| `null`        | 空 |
| `undefined`   | 未定义 |
| `unknown`     | 未知 |
| `never`       | 永不存在 |

## 常量 & 变量

  `TypeScript` 中的数据的定义
  
```ts
// 定义常量
const PI: number = 3.1415926;
// 定义变量
let no: number = 42;        // 整形变量
let name: string = "tom";   // 字符串变量
name = "peter";
```
