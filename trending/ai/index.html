                  # Transformer vs RNN vs CNN

                  • Transformer : (Multi-Head Attention) 多头自注意力机制

                    特色 : 自然语言处理

                  • RNN : 循环神经网络

                    特色 : 处理序列数据和时序关系

                  • CNN : 卷积神经网络

                    特色 : 处理网格数据(如: 图像处理)
