# 前端优化

前端程序来说，由于是和用户交互，这些优化不仅涉及程序的本身的优化，还涉及一个用户体验方面的优化

# 启动优化

也就是打开程序的快慢。 App 角度来说就是用户点击打开 App 到看到首屏展示的速度。 Web 角度来说就是用户打开网站到网页加载完成。

  • 减小包体积。
  • 减少启动逻辑，将启动无关的逻辑延迟加载。
